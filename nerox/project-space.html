SPACE TOURISM WEBSITE

Home Space Tourism

Responsive Front End Project Space Tourism Website

  • Tools: Pure HTML, CSS, and JavaScript

  • Features: Responsive, Accessible, Design System, Utility Classes

Replicate the Design from Figma Files

Front End Mentor Challenge

Use the Site

Access the live site, and plan your trip to space! View it on different devices or resize your window to see how responsive the site is and how good it looks regardless of screen size. If you have a screen reader, use it to see the accessibility features built into the site. Use the "tab" key to see how a user without a mouse can still navigate the site.

Project Brief

Given the Figma design files, create the Design System and website from scratch. Ensure the site is responsive so it's readable on any device, viewport, or screen size. Ensure the site is accessible to users with screen readers and users without a mouse.

Bonus Challenge

Additionally, I challenged myself to only use pure HTML, CSS, and JavaScript to further my foundational understanding of these languages and their inner workings instead of using frameworks or libraries.
